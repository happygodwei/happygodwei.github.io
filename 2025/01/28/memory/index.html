memory

发表于2025-01-28|更新于2025-01-28
|阅读量:

魔非魔道非道善恶在人心,欲非欲情非情因缘由天定

小学

我和她是小学认识的。我的小学一共分过两次班，四升五一次，五升六一次。就是四年级升五年级，我们相遇了。小学的记忆已经模糊，只能依稀记起一些往事。我不记得我是什么时候开始喜欢她了，但我知道我从小学就喜欢她了。我和她回家是顺路的，偶尔会和她一起回家，有一天她问我你有喜欢的人吗？还一直问我是谁，我说QQ告诉她，当时我是有手机（智能机）的，但是我手机没有流量，当时还有QQ网页版，我就用老年机登QQ（消耗流量少）去回复她，我给她发其实我喜欢你，发了好几遍，我没收到回复，第二天我就去了北京我大舅家，我用我妈妈的手机登上QQ，给她发了一句我喜欢你，一直到最后我也没收到回复。这件事后，好像对我们没有任何影响，就像我没发过消息一样。就这样日复一日，五年级的时光就过去了。到了六年级，又分班了，我们不在一个班，不在一个班我有时候想和她放学一起走需要创造“偶遇”的机会。就这样小学生活结束了。上初中了，我感觉我可能再也见不到她了，但可能是缘未了吧，初中我们又相遇了。

初中

在初中报道那天，看分班表，我有个好兄弟对我说xxx和我在一个班。当时我以为是同名的人不是她（我五年级的时候就有和她同名的同学），直到到了教室我才发现真的是她。我和她不仅在一个学校，而且还在一个班，我和她的故事就这样续写了。

补充一下细节吧，补充一下学习成绩，我小学学习成绩不算高，只有数学学的比较好，基本能考满分，语文英语深圳可能会不及格。她当时学习就是神，所有科目都很高。小学毕业考试我看了我们市的成绩单，她能排前5，我排在中间，大概130多名（小学300人左右）。初中的时候她学习也比我要好得多，我入学是两百名，她就是前几名。

在初中，我和她每天放学都一起回家，关系也日渐升温。记得初中我有一年生日，她送了我一个生日礼物，她亲手制作的，用一个爱心盒子装着，里面有她亲手折的纸船上面写着祝福的话，除了这些，里面还有好多好多东西……我当时好像就只说了句谢谢，没有任何其他的回应。有一次办运动会，我们玩真心话大冒险，到她说真心话，我有个兄弟问她什么时候和我搞对象，她说如果在一个高中就和我在一起，当时我就想一定要和她考一个高中。有一次音乐课下课，她好像又是玩真心话大冒险，输了选大冒险，她的惩罚是抱住我，当时她抱住了我，我的脑子一片空白，不知所措。

中考结束我和她在QQ表白，她拒绝了，她说她不想搞对象。可能我让她失望了吧，我感觉我也配不上她。

高中

在高中之前，我们每个假期都会一起看一场电影。高中以后好像一切都变了。
高中我们还在一个学校，我和她高中都是被同一所中学同一个校区录取。当时我家长想找人把我送到另一个校区，我说没必要，不用换校区，但是我家里人不听把我的校区换了，我当时伤心不已。可能还是缘未了，她的家里也给她换了校区，我们两个都换了校区，又在一个学校了，可惜的是她是实验班的大佬，我是普班的普通学生。
上了高中，我依旧那么怂。还记得我和她一起在食堂吃饭，还记得她给我夹的那块鸡叉骨。但我和她一共在食堂吃过几次饭？是我每次在食堂寻觅着她吗？期待着每一次的“偶遇”。但高中我和她好像渐行渐远了，之前的我们周末和假期有时间会一起看电影，一起出去玩，可是高中为什么不一起出去了呢？我也不知道为什么。到现在也想不明白。有一次晚上，我忍不住在QQ给她发了一句“想你了”，记得她也回了一句“我也是”。可这终究有什么用呢？高三的时候，在晚饭期间，我经常去操场和不同的人谈人生聊理想。我在三楼，她在一楼，我经常下楼去她们班门口看一眼，看看她在不在教室，在教室的话我就可以和她一起在操场上散步聊天，但我和她在操场聊天的次数屈指可数，每次和她在操场散步聊天我都会高兴整个晚自习。

大学

上了大学，我和她渐行渐远，她在福建，我在天津，相隔千里。为什么？为什么？为什么我在高考后没有和她表白呢？我到现在也想不明白。在去上大学的前一天晚上，我们在一起散步，走了一个晚上，谈天说地，我却失去了勇气，将本要说的话藏在了心中。在上大学的时候，我们在QQ约好了要在假期时出去玩，但在假期为什么没有去，是都忘却了吗？我没忘，但为什么没去？为什么？
